# Getir
Getir Assignment



Open the project and install node dependencies.

**With NPM**
```shell
$ npm install 
```


Below command will start the node server

```shell
$ npm run dev
```

If don't want to start from npm script, use below command to start all with docker
```shell
$ docker-compose up
```

For testing run below command
```shell
$ npm run test
```
